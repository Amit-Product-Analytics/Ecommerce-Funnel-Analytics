# E-Commerce Funnel Analytics Dashboard

## Project Overview

This project analyzes the customer purchase journey from website visit to completed purchase.

The objective of this project is to identify funnel drop-offs, understand customer behavior, optimize conversion performance, and improve revenue generation.

---

## Business Problem

E-commerce businesses need to understand where customers leave the purchase journey and what factors influence conversion.

This project answers the following business questions:

* What is the overall conversion rate?
* Where do users drop off in the funnel?
* Which traffic source performs best?
* Which product category generates the most revenue?
* Which cities achieve the highest conversion rate?
* How can conversion performance be improved?

### 5. Business Insights

Analysis:

* Funnel Findings
* Conversion Bottlenecks

* Revenue Opportunities

---

## Key Insights

* Only 44 out of 200 visitors completed a purchase.
* Organic Traffic achieved the highest conversion rate.
* Social Media generated the highest revenue.
* Home Decor generated the highest revenue.
* Pune recorded the strongest conversion performance.
